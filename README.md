# shaka-packager-pssh

Dockerfile to setup the environment to work with the Shaka Packager pssh tool

# Usage

```
docker run -e PROVIDER='yourProvider' -e CONTENT_ID='yourHexContentId' -e KEY_ID='yourHexKeyId' gfronza/shaka-packager-pssh
```
